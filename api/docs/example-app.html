Example Android Application

The repository includes an example Android app demonstrating SDK initialization, configuration, and ad playback behavior.

Run Locally

  1. Open the example app in Android Studio.
  2. Ensure internet access is enabled (device/emulator).
  3. Initialize the SDK with the Render URL.
  4. Trigger ads using clicks or interval mode.

Sample Integration

AdsSdk.init(
  context = applicationContext,
  baseUrl = "https://video-ads-sdk.onrender.com/",
  appId = "demo_app"
)
