Admin Portal

The Admin Portal is a static HTML page served by the API service. It allows managing ads per app_id and interacts with the backend using REST calls.

Access

https://video-ads-sdk.onrender.com/admin

Capabilities

  • View existing ads for an application
  • Create new ads using video_url (MP4)
  • Delete ads
