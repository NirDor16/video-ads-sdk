Example Android App

The example app demonstrates how to integrate the SDK and verify that ads appear on user interaction (click-based) or time interval (interval-based).

Integration steps

  1. Add the SDK dependency (JitPack).
  2. Call AdsSdk.init(...) once at app startup.
  3. Optionally call setPreferences() to override trigger & categories.

Recommended setup (MainActivity)


AdsSdk.init(
    context = applicationContext,
    baseUrl = "https://video-ads-sdk.onrender.com/",
    appId = "demo_app"
)

lifecycleScope.launch {
    AdsSdk.setPreferences(
        categories = listOf("TV", "CAR", "GAME"),
        triggerType = "CLICKS",
        clicksCount = 5,
        xDelaySeconds = 5
    )
}

What to expect

  • After enough clicks (CLICKS mode), a full-screen video ad appears.
  • The close (X) button appears after the configured delay.
  • If target_url exists, clicking can redirect; otherwise no action.
