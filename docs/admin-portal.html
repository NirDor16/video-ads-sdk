Admin Portal

The Admin Portal is a simple web UI for managing ads in MongoDB via the backend API. It supports listing existing ads, creating new ads, and deleting ads.

How to open

If your backend runs on Render, open: https://video-ads-sdk.onrender.com/admin

The portal uses App ID to manage content per application.

Features

  • Load Ads ‚Äî view all ads for a specific app_id
  • Upload / Create ‚Äî add a new ad (URL-based)
  • Delete ‚Äî remove an ad permanently

Why URL-based?

For this project, ads are stored as URLs in the database (video_url). This keeps the backend simple and cloud-friendly.

  • You can host MP4 files anywhere (CDN, public storage, etc.).
  • No need to upload files to the backend server.
